# SinaSpeedStatus

| Symbol | Value | Description |
| ------ | ----- | ----------- |
| NO_FAULT | `WORD#16#7002` | No fault active |
| DRIVE_FAULT_ACTIVE | `WORD#16#8401` | Drive fault active |
| ON_INHIBIT_ACTIVE | `WORD#16#8402` | Switching on of drive is inhibited |
| READ_ERROR | `WORD#16#8600` | Error reading data from drive |
| WRITE_ERROR | `WORD#16#8601` | Error reading data from drive |
